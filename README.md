# 🎬 Movie Management System

## 📌 Description

A Python-based application using SQLite to manage movies with CRUD operations.

## 🚀 Features

* Add, View, Update, Delete movies
* Search by genre
* Sort movies
* Count movies by genre
* Input validation and error handling
* Indexed search for better performance

## 🛠️ Technologies Used

* Python
* SQLite

## ▶️ How to Run

1. Install Python
2. Run:

   ```
   python main.py
   ```

## 📷 Sample Output

Menu-driven CLI application to manage movie records.
